# Fast Tasks

## Purpose

This folder stores lightweight recurring tasks that are too specific to be a
general rule and too small or situational to justify a dedicated skill.

Fast tasks are activated only by exact user messages documented here. If the
message contains extra words, treat it as a normal request and use the standard
read order and skill selection.

## Adding A Fast Task

For each task, document:

- exact trigger message
- purpose
- steps
- constraints
- expected response shape

Keep fast tasks short and operational. If a task becomes broadly reusable or
requires substantial workflow guidance, move it into `.agents/skills/`.
